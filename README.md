# 大学生志愿服务场域 ABM 仿真模型

基于多智能体仿真模拟（Agent-Based Modeling, ABM）的志愿服务场域研究。

## 项目概述

本项目构建了大学生志愿服务场域的 ABM 模型，基于 572 名志愿者的实证调查数据，
通过 6 组反事实政策实验评估组织管理的关键机制。

**核心发现：**
- 学业挫折感（期望-现实差距）是导致志愿者流失的首要因素
- 缩减招募规模（300→200人/年）可使学年流失率降低约 23%，晋升率提升 7.8 个百分点
- 增加活动经费主要改善组织质量（优秀项目占比提升），而非防止流失

## 模型机制

规则 1：9 月招募（大一新生，年学业压力=0.30）
规则 2：月度参与意愿（满意度×0.45 + 学业压力×0.35 + 同伴效应×0.20）
规则 3a：满意度更新（服务质量体验）
规则 3b：挫折感更新（期望-现实差距，独立于满意度）
规则 4：**三机制月度流失**（满意度 + 挫折感 + 学业压力，统计独立）
规则 5：权力更替（年度管理层换届）
规则 6：项目创新（管理层驱动，成功率受经费和时间调节）
规则 7：年度激励分配（激励阈值=20）
规则 8：届际更替（association_years≥2 强制退出）
规则 9：行为异质性（管理能力、正向/负向倾向随机分布）

## 核心参数

| 参数 | 值 |
|------|-----|
| 初始成员 | 849 人 |
| 管理层人数 | 93 人 |
| 年度招募 | 300 人/年 |
| 月度流失-满意度系数 | 0.025 |
| 月度流失-挫折感系数 | 0.010 |
| 月度流失-学业压力系数 | 0.025 |
| 模型校准 | 学年流失率=42.8%，管理层=90.7 人 |

## 运行方式

```bash
# 运行 6 组实验（15 次蒙特卡洛重复，~30 秒）
python run_exp.py

# 生成 6 张图表
python run_figures.py

# 编译研究报告
xelatex report.tex
```
