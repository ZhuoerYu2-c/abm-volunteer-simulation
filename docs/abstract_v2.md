# 论文写作草稿：摘要与引言

## 摘要（待整合）

高校志愿服务场域普遍呈现"入口宽、出口窄"的结构性困境：大量新生涌入，但高年级学生持续参与率偏低，年级抑制现象显著。本研究整合场域理论与行动—结构互构框架，利用572份大学生志愿者问卷数据进行参数校准，构建了基于多智能体仿真（Agent-Based Modeling, ABM）的志愿服务场域动态演化模型。研究识别出学业压力、挫折感（期望—现实差距）与满意度（服务质量体验）三个机制独立驱动成员流失，并通过8组蒙特卡洛仿真实验系统检验了不同干预策略的效应与机制。结果表明：（1）学业压力与挫折感构成流失的双轨驱动机制；（2）减少招募规模是最具杠杆效应的单一干预手段，可使学年流失率从43.0%降至34.6%；（3）激励政策存在非线性区间效应（服务量下限与上限），简单提高强度或降低门槛的边际效用有限；（4）取消届际更替限制呈现"短期改善、长期恶化"的逆转模式，反映了行动—结构互构的动态复杂性。本研究推进了场域理论在计算社会科学中的可操作化，为志愿服务组织的弹性管理提供了模拟参考。

## 引言要点

### 研究背景
- 志愿服务是青年社会化的重要实践空间
- "入口宽、出口窄"的困境：招募可观但持续率低
- 年级抑制现象在高校志愿服务中普遍存在
- 现有研究多采用静态方法，难以处理动态交互与非线性效应

### 理论gap
- 场域理论提供了理解行动—结构互构的框架，但缺乏计算可操作化
- ABM在政策实验中有优势，但与实证数据的结合不够紧密
- 志愿服务流失研究缺乏对三机制独立效应的系统检验

### 方法选择
- ABM能够捕捉时间维度的积累过程
- ABM能够处理非线性动态与自我强化机制
- ABM支持反事实政策实验

### 研究问题
1. 学业压力、挫折感与满意度三个机制如何独立与交互地驱动成员流失？
2. 不同干预策略的效应与机制有何差异？
3. 干预策略之间是否存在协同效应或替代效应？

### 贡献
1. 理论：推进场域理论从定性描述向计算可操作化的转换
2. 方法：展示如何将问卷调查数据与ABM相结合
3. 实践：为志愿服务组织提供量化的政策实验工具
