## 🔴 数据质量风险说明
====================================================================
1. 572行原始数据中139种记录各精确重复4次，此模式高度规律，
   很可能源于数据录入或SPSS导出过程，而非自然的面板数据。
2. 142条唯一记录代表了真实的受访者模式。
3. 均值、比例等描述性统计以原始572行报告，结果不变。
4. 回归分析使用142条唯一记录，标准误和显著性需审慎解读。
5. 原始报告"β=-0.151, p<0.05"的"管理层保护效应"目前无法复现，
   可能原因：(a)原始分析使用了不同的管理层定义；(b)使用了加权处理；(c)数据处理方式不同。
